## 判据链

1. **内容严重程度 = 低危**：盗刷指控本身敏感，但完全无证据+全大写情绪化+模板化语言 → 降为低危
2. **传播影响力 = 零**：新账号(2周)+粉丝8+零互动
3. **严重度 = P3**：低危 × 零影响力

## 边界讨论

**最敏感判断：为什么"盗刷指控"不自动升级为 P1？**

盗刷指控如果属实且有传播 → 应触发 P1/P0。但此案例完整信号组合指向水军：
- 新账号 + 只有同类推文 + 零粉丝 + 全大写 + 模板化语言 + 零互动

如果判断错了（真实用户）——损失几乎为零（无传播）。
如果每条类似内容都升级——系统将被噪声淹没。

**升级触发条件**：
- 单条 → P3 可忽略，但记录归档
- 24h内≥5条同类 → P1 有组织攻击
- 已有传播(≥1000赞) → 按影响力评级，不因虚假信号降级

## 对标注规范的影响

确认**真实性评估与严重度的交互规则**：
1. "大概率虚假 + 零传播" → 可降级（P2→P3）
2. "大概率虚假 + 有传播" → **不能降级**
3. "存疑 + 敏感内容" → 保持评级不变，持续观察
4. 永远不能仅凭真实性判断决定忽略——必须同时考虑**传播量**

这一条来自 DJI 经验的直接映射：DJI 系统中的"是否真实"是辅助字段，不影响系统 classification 的自动生成。
