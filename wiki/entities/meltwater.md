# Meltwater

## 概述

Meltwater 是全球领先的媒体智能与舆情监测 SaaS 平台。在 TEMU 的舆情体系中，Meltwater 承担海外舆情监测的主力工具角色，与国内的[[entities/sina-yuqingtong|新浪舆情通]]形成互补。

## 在舆情体系中的位置

```
海外舆情监测：Meltwater（主力） + 人工搜索（补齐）
国内舆情监测：新浪舆情通（主力） + 人工搜索（补齐）
```

## 核心能力

- 跨平台实时监测（新闻、社媒、博客、论坛）
- 关键词/布尔逻辑搜索
- 情感自动分析
- 影响力评估（到达率、互动量）
- 报告与告警

## 在 AI 替代评估中的位置

Meltwater 本身已经是自动化工具。AI 的价值不在于替代 Meltwater，而在于：
1. **替代人工搜索补齐**——Meltwater 覆盖不到的长尾内容
2. **增强 Meltwater 输出的分析**——自动打标、分流、摘要
3. **连接 Meltwater 和下游系统**——自动从监测到分流的端到端链路

即：Meltwater 解决"采集"，AI 解决"理解+决策"。
