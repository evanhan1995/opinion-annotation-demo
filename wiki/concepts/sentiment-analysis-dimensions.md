# 多维度情感分析

## 概述

不同于简单的"正面/负面/中性"三分法，多维度情感分析要求在**品牌、产品（可多个）、竞品**三个维度上分别判断情感倾向。此方法直接来源于 DJI 舆情系统的标注体系设计。

## 为什么要多维度？

| 场景 | 单维度判断 | 多维度判断 |
|------|-----------|-----------|
| "DJI售后很好，但Mini 4 Pro太差了" | 混合（无法区分） | 品牌正面 + 产品负面 |
| "GoPro比DJI强多了" | 负面 | 可能为中性评测 |
| "用DJI Pocket 3拍的，不评论产品" | 中性 | 品牌无关 + 产品无关 |

## 三维分析框架

### 第一维：整体情感（单选）

| 标签 | 定义 |
|------|------|
| **正面** | 推荐、赞美、感谢倾向 |
| **负面** | 批评、抱怨、警告倾向 |
| **中性** | 客观陈述，无评价语言 |
| **混合** | 同时包含明显正负面 |

### 第二维：品牌维度

关注**品牌整体声誉**，不限于具体产品。
- **正面**：赞扬品牌理念、企业责任、服务态度
- **负面**：攻击品牌形象、质疑企业诚信
- **无关**：不涉及对品牌本身的评价

### 第三维：产品维度（可多产品）

识别内容中提及的具体产品线/型号，逐产品判断情感。
- 明确型号→精确匹配；模糊品类→归类品牌整体；无法判定→品牌整体
- **正面**：使用满意度、功能赞扬、推荐购买
- **负面**：功能缺陷、质量不满、使用困扰

### 第四维：竞品维度

当内容提及竞品时独立标注。来自 DJI 系统设计。
- 是否提及竞品 + 竞品情感 + 与我方对比（有利/不利/中性）
- **区分规则**：公正评测（有褒有贬、结论不极端）≠ 负面舆情；恶意对比（踩一捧一、煽动性）→ 按负面评级

## AI 标注常见陷阱

1. **关键词过拟合**："scam""垃圾"等词不一定代表严重负面（可能是夸张语气）
2. **混合情感误判为负面**：提及不足 ≠ 整体负面
3. **竞品提及误判为竞品攻击**：仅提及竞品名 ≠ 负面
4. **品牌与产品情感混淆**：产品负面 ≠ 品牌负面
