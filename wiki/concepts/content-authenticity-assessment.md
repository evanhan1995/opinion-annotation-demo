# 内容真实性评估

## 概述

评估一条舆情内容是否为真实用户的真实体验。来自 DJI 舆情系统中的"是否真实(Y/N)"字段。这是**辅助维度**——不应直接影响严重度评级，但可影响分流策略。

> 置信度标为 medium：真实性判断是舆情分析中最困难、AI 最不可靠的环节。高级水军可以骗过大多数自动化检测。

## 真实性信号清单

### 真实信号
1. 具体使用细节（时间、地点、场景）
2. 可验证信息（订单号、序列号、对话截图）
3. 账号历史丰富、内容多样、时间跨度>6个月
4. 互动模式自然（非集中性点赞）
5. 语言风格个性化（非模板化）
6. 态度理性（有分析、有对比）

### 虚假信号
1. 账号异常新（<3个月），历史内容单一
2. 同质化文案（多账号相似内容）
3. 集中爆发（同主题短时间内大量出现）
4. 模板化语言（缺少具体细节、全大写/大量感叹号）
5. 账号与水军团伙有关联
6. 极端情绪无实质信息

## 判断输出

| 判断 | 条件 |
|------|------|
| **大概率真实** | ≥3 真实信号，无虚假信号 |
| **存疑** | 真实和虚假信号各有一些 |
| **大概率虚假** | ≥2 虚假信号，无强真实信号 |
| **无法判断** | 信息不足 |

## 与分流的关系

**核心原则**：真实性权重 < 传播影响力权重。

| 真实性 | 传播量 | 分流影响 |
|--------|--------|---------|
| 大概率虚假 | 零传播 | 可忽略的辅助依据 |
| 大概率虚假 | 有传播 | **不能降级**！假消息也能造成真损害 |
| 存疑 | 任何 | 不影响正常分流 |
| 大概率真实 | 任何 | 不自动升级 |

> 从信息论角度：信息的影响熵与信息的真实性是独立的。假消息在传播量大的时候，其舆论影响与真消息无异。

## DJI 系统的设计启示

DJI 系统中"是否真实"是独立辅助字段——系统不会因"判定为假"而自动降低 Classification 等级。这个设计是深思熟虑的。

## AI 标注常见陷阱

1. **过度依赖账号年龄**：新账号 ≠ 假账号
2. **情绪化 ≠ 虚假**：真实用户愤怒时也会全大写
3. **模板化语言的语境依赖**：热门话题下跟风使用相同措辞≠水军
4. **无法判断时强行判断**：标"无法判断"比错误判断更安全
